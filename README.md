# MadsNotes

A lightweight desktop study tool for turning PDFs and text into clean, editable study notes.

Designed for fast reading, revision, and browser-based language learning workflows.

---

## Download

1. Open the **Releases** section
2. Download the latest `.zip`
3. Extract the archive
4. Run `MadsNotes.exe`

No installation required.

> If Windows shows a security warning:
> Click **More info → Run anyway**

---

## Features

* Paste text directly into the workspace
* Upload or drag-and-drop PDFs
* Automatic PDF text extraction
* Editable study workspace
* Generate clean browser-readable notes
* Save and organize study notes
* Search saved notes
* Reopen previously generated notes
* Theme support
* Fully offline

---

## Workflow

### PDF Workflow

```text
Upload PDF
→ Notes generated automatically
→ Browser opens
→ Edit if needed
→ Save Note
```

### Text Workflow

```text
Paste Text
→ Generate Notes
→ Browser opens
→ Save Note
```

## Notes
