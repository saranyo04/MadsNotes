# Mad's Chinese

A simple desktop tool to convert Chinese text and PDFs into clean, readable study notes. Optimized for use with browser extensions like Zhongwen.

---

## Download

1. Go to the **Releases** section of this repository  
2. Download the latest `.zip` or `.exe` file  
3. Extract the zip (if needed)  
4. Double-click `MadChinese.exe` to run  

No installation required.

> If Windows shows a security warning:  
> Click **More info → Run anyway**

---

## What It Does

Mad's Chinese helps turn messy Chinese input into structured, editable notes and generates a clean HTML page for easy study on a laptop.

---

## Features

- Paste Chinese text
- Upload or drag-and-drop PDFs
- Automatic text extraction from PDFs
- Clean structured output:
  - headings  
  - lists  
  - labels  
  - paragraphs  
- Built-in editor to fix formatting
- Generates clean HTML notes
- Opens in browser for Zhongwen hover lookup

---

## Notes

- Works fully offline  
- No accounts or setup required  
- Use browser extensions (like Zhongwen) for word lookup
