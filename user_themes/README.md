# User Themes

Drop custom theme JSON files in this folder. User themes override shipped
themes with the same `name`. Each file should define the required tokens:

```json
{
  "name": "My Theme",
  "primary": "#F6C1C8",
  "secondary": "#B7D7A8",
  "background": "#FFFDF9",
  "surface": "#FFF9F4",
  "text": "#4A4A48",
  "accent": "#D2475C"
}
```
